# Safe Miles

Safe Miles is a school transportation optimization system made to improve how districts assign bus routes by accounting for student safety, real road conditions, and operational constraints. 

Safe Miles computes walking eligibility using a pedestrian road network built from OpenStreetMap data and calculates shortest path walking distances. Students exceeding a configurable safe walking threshold, or those without a safe pedestrian path, are assigned to bus routing. Bus riders are then grouped into stop locations using K-Means clustering, which is refined using a maximum-distance constraint that recursively splits clusters when any student exceeds a safe walking radius from their assigned stop. These stops are snapped to real road network nodes, and a full distance matrix is computed using Dijkstra based shortest path calculations over the road graph. Final bus assignments are generated by solving a Vehicle Routing Problem (VRP) with capacity constraints and fleet limits using OR-Tools, ensuring each route respects bus capacity while minimizing total travel distance. Beyond creating bus routes, Safe Miles is a tool that can be used by administrators and students. For administrators, Safe Miles computes route-level and system-level metrics such as bus utilization, load distribution, and total distance traveled. It allows for quick creation of multiple routes that respect different constraints. Admins are also able to see organized student feedback about the route plans. For students, Safe Miles explicitly displays their bus stop, bus route, safety tips, a feedback form, and a simulated bus tracking feature.

Overall, Safe Miles improves safety aware routing, simplifying the workflow for administrators and providing a safer and better transportation experience for students. This optimization system improves walkability decisions, bus stop placement, and route efficiency using real geographic and constraint based modeling.
